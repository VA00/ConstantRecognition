# Targets (What to recognize)

This module contains algorithms and data structures corresponding to the target of the search.

## Modules
- `constant`: Single constant recognition (e.g., Pi, e)
- `multiple_constants`: Searching for expressions combining multiple specific unknown constants
- `function`: Identifying mathematical functions (e.g., f(x) = sin(x) + c)
- `sequence`: Identifying sequences (e.g., integer sequences, OEIS-style)
